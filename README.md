# sm64hack-archipelago-jsons
Simple repository of all current working JSON files for sm64 romhacks

If you want the randomizer itself, it can be found [here](https://github.com/DNVIC/archipelago-sm64hacks/)

If you've made a JSON file and want to share it here, either message @DNVIC on discord with the JSON file, or make a pull request, and I'll add it.

JSON Author credit and other JSON info is now on this google sheets page (maintained by Iwer Sonsch!) - https://docs.google.com/spreadsheets/d/1zr2Fw4MJED8i3f-tmnc7jQvsOrteTIT2nPLTEzN9slo/edit?gid=0#gid=0
If you want more extensive info about hacks, it can be found here on the wiki - https://wiki.dnvic.com/index.php/Main_Page
